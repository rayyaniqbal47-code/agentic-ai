# Agentic AI Chat (n8n Workflow)

This is a simple AI chatbot workflow built using n8n.  
It uses Google Gemini AI to respond to user messages and can also interact with Google Sheets and Gmail.

---

## Features

- AI chatbot using Google Gemini
- Memory to remember previous messages
- Read data from Google Sheets
- Send emails using Gmail
- Webhook-based chat trigger

---

## Tools Used

- n8n
- Google Gemini AI
- Google Sheets
- Gmail API

---

## How It Works

1. User sends a message to the chatbot
2. AI Agent processes the message
3. It can:
   - Reply using AI
   - Get data from Google Sheets
   - Send an email using Gmail
4. Response is sent back to the user

---

## Setup

1. Import this workflow into n8n
2. Add your Google Gemini, Sheets, and Gmail credentials
3. Activate the workflow
4. Send a message via the webhook

---

## Use Case

This workflow can be used as:
- A simple AI chatbot
- Email automation assistant
- Data lookup assistant
